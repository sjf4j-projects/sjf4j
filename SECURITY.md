# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in SJF4J, please do **not** open a public issue. Instead, report it privately so that it can be addressed securely.

Please send an email to: hannyu@gmail.com

Include the following information in your report:

- Detailed description of the vulnerability
- Steps to reproduce
- Version(s) affected
- Any suggested mitigations (optional)

You will receive a response within 3 business days.

## Supported Versions

All versions > 1.0.0


## Security Update Policy

- Once a vulnerability is reported, the team will acknowledge it and create a patch or release within a reasonable time frame.  
- Security fixes will be released as soon as possible and may be backported to supported versions.  
- All security releases will be clearly marked in the changelog.

---

Thank you for helping keep SJF4J safe!
